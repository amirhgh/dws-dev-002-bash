# Try

Try your script helps you to try your command any times you like.

### Usge

Use `-i` for interval in Secods and use `-n` for number of try.

you can use local env whith these varialbes:

```bash
TRY_NUMBER=12
TRY_INTERVAL=5
TRY_COMMAND=NULL
```

```bash
try -i 5 -n 12 which bash
```
